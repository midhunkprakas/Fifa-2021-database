# Fifa-2021-database
Exploratory data analysis of FIFA 2021 player dataset to gain insights into player performance, attributes, market value, and positions using data analytics techniques.
This project focuses on analyzing the FIFA 2021 players dataset to uncover insights about player performance, attributes, positions, overall ratings, potential, wages, and market value. The analysis helps understand trends in player skills across different clubs, leagues, and nationalities.

The project uses data analysis techniques to clean, explore, and visualize the dataset, making it useful for football analytics enthusiasts and aspiring data analysts.
